This directory contains the files of the first-order neighbor structure of all regions.

This information is exclusively geographic information and is common and independent of each of the seasons. The data in the columns are as follows:

The first column corresponds to the number of neighboring sites of the site located in the specific row. The following columns show these (first-order) neighbor sites. Each site has a maximum of four neighbors.
